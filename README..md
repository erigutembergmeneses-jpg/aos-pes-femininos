## 🌐 Sobre o Projeto

**Aos Pés Femininos** é um livro-álbum que reúne 12 versões musicadas de um mesmo soneto, cantado em 12 idiomas com o auxílio de inteligência artificial. Cada faixa respeita a prosódia, a métrica e a alma de cada língua.

## 🎵 Características

- **12 idiomas**: Românico, Germânico, Eslavo, Helênico e Japonês
- **Gêneros musicais variados**: Do fado ao honkyoku
- **IA Generativa**: Usada como ferramenta paramétrica
- **Material completo**: Livro digital com análises, prompts e glossário

## 🚀 Como Usar

1. Faça o download de todos os arquivos
2. Coloque a imagem `capa-aos-pes-femininos.jpg` no mesmo diretório
3. Abra o `index.html` no navegador ou faça o deploy no GitHub Pages
