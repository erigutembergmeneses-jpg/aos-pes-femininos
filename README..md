# 🎼 Aos Pés Femininos

### *Quando a Poesia Clássica Atravessa Fronteiras Linguísticas*

> *"A melodia sob seus pés — escaneie para ouvir o eco dos passos que inspiraram esta coleção."*

---

## 🌿 Sobre o Projeto

**Aos Pés Femininos** é um livro-álbum experimental que reúne **12 versões musicadas** de um mesmo soneto, cantado em 12 idiomas com o auxílio de inteligência artificial generativa. Cada faixa respeita a prosódia, a métrica e a alma de cada língua — do português ao japonês, do russo ao grego.

O projeto explora a interseção entre:

- 📜 **Poesia clássica** (a forma soneto, com mais de 700 anos de tradição)
- 🎵 **Musicologia comparada** (modos gregos, escalas pentatônicas, prosódias específicas)
- 🤖 **IA generativa** como intérprete paramétrica (não como autora)

Todo o material — livro digital, faixas de áudio e materiais complementares — está disponível gratuitamente no [**Internet Archive**](https://archive.org/details/pieds-feminins), sob licença Creative Commons.
